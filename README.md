# 💰 Expensio - Basic Expense Tracker

A simple expense tracker application created using **HTML, CSS & JavaScript**.
Helps users to keep track of their income and expenditure easily.

---

## 🚀 Features

* Add income and expenditure
* Delete transactions
* Clear all data
* Auto-calculated balance
* Save data locally (in browser localStorage)
* User-friendly interface

---

## 📁 Folder Structure

```
expensio/
├── index.html
├── style.css
└── script.js
```

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* LocalStorage

---

## ▶️ Usage

1. Clone/download project
2. Open `index.html` in any web browser

---

## 💡 Logic

* User adds a transaction
* Saves data in localStorage
* Calculation of income/expenditure
* Balance = income - expenditure
